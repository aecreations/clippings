Clippings 6 – Jetzt verfügbar.

Clippings 6 ist eine WebExtension, die mit Firefox 57 und neuer kompatibel ist. Sollten Sie eine Aktualisierung von Firefox planen, sollten Sie sich auch auf die Aktualisierung von Clippings auf Version 6 vorbereiten.

Vorbereitungen

Sobald Clippings auf die neue WebExtension aktualisiert wurde, gibt es einen zusätzlichen Schritt, den Sie ergreifen müssen, um Ihre Textbausteine zu erhalten. Um das Upgrade vorzubereiten, sollten Sie eine Sicherheitskopie Ihrer Textbausteine erstellen.

Tipp: Sie können Ihre Textbausteine beliebig oft, bis Clippings auf Version 6 aktualisiert wird, sichern.

Um sicherzustellen, dass Sie über eine so aktuell wie mögliche Sicherung verfügen, sollten Sie dies regelmäßig machen.

Clippings sichern

  1. Textbaustein-Verwaltung öffnen.
  2. Klick auf Optionen → Datensicherung.
  3. Wählen Sie einen Speicherort auf Ihrem System aus, in den exportiert werden soll (zb. Desktop oder Ihren Ordner "Dokumente").

Stellen Sie sicher, dass Sie sich merken, wo Sie die Sicherung gespeichert haben. Sobald Clippings aktualisiert wurde, wird die Sicherungsdatei benötigt.

» Zum Öffnen der Textbaustein-Verwaltung hier klicken um jetzt eine Datensicherung zu erstellen

Nach Aktualisierung auf Clippings 6

Nach Abschluss der Aktualisierung, folgen Sie den Bildschirmanweisungen um Ihre Textbausteine in Clippings 6 zu integrieren.
Anschließend sind Sie wieder startbereit.
